# Build a simple RAG from scratch

## Step 1. config modify：
1. .env 中的 api key
2. data 文件夹下的测试数据

## Step 2. activate environment
``` shell
conda create -n simpleRAGEnv python=3.8
pip install -r requirements.txt
```

## Step 3. run the project
``` shell
python main.py
```
