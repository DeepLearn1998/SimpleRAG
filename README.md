# 零框架纯手搓 RAG

## Step 1. 替换为自己的内容：
1. .env 中的 api key
2. data 文件夹下的测试数据

## Step 2. 安装虚拟环境
``` shell
conda create -n simpleRAGEnv python=3.8
pip install -r requirements.txt
```

## Step 3. 启动项目
``` shell
python main.py
```
