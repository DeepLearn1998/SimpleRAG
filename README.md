# Build a simple RAG from scratch

## Step 1. modify config
1. 'api key' in .env
2. test dataset from data folder

## Step 2. install requirements
``` shell
conda create -n simpleRAGEnv python=3.8
pip install -r requirements.txt
```

## Step 3. run the project
``` shell
python main.py
```
